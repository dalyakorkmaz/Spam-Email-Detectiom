Spam Email Detection (Naive Bayes)
This project is a machine learning-based spam detection system developed using Python.  
It classifies SMS/email messages into two categories:

- Spam (1)
- Ham (0)

The model is built using Natural Language Processing (NLP) techniques and the Naive Bayes algorithm.

Technologies Used

- Python
- Pandas
- Scikit-learn
- CountVectorizer (NLP)
- Multinomial Naive Bayes

---

Dataset

- SMS Spam Collection Dataset
- Contains labeled messages as spam or ham

How It Works

1. Dataset is loaded and cleaned
2. Labels are converted into numeric values
3. Data is split into training and testing sets
4. Text data is converted into numerical vectors
5. Naive Bayes model is trained
6. Model makes predictions on test data
7. Performance is evaluated
